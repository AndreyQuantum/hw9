# Задание
Добавьте к сервису CRUD из предыдущего домашнего задания новый маршрутизатор /auth, который содержит эндпойнты:

1) регистрация; 
2) вход (аутентификация) пользователя; 
3) выход (завершение сессии).
С этого момента незарегистрированные пользователи не должны иметь доступ к другим эндпойнтам вашего приложения.

Настройте тип авторизации (на выбор):

1) по идентификатору (базовая сложность);
2) *по user_token (повышенная сложность);
3) ** по user_token/refresh (высокая сложность).
4) *** Настройте для некоторых пользователей права только на чтение (необязательно, высокая сложность).
